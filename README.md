# Instructions

## Profile Add/Update

If you want to add or update your profile, you have two options:

- Visit https://mode-demo.github.io/profile-submission.html and fill in all the required information. Your profile will be added soon.
- Alternatively, fork this repository and create a new folder with your name (e.g., `Hamza-Hanif`). Inside that folder, add two things:
  - Your profile information in an `_index.md` file  
  - Your profile picture, renamed as `avatar.jpg`, `avatar.jpeg`, or `avatar.png`  

  Then push your changes and create a PR/MR. Hamza or Pietro will merge it.

---

## News

- Fork this repository, make changes in `content/news/index.md` (there is a commented example showing where to add items), then push and create a PR/MR. Hamza or Pietro will merge it.

---

## Opportunities

If you want to share PhD/Postdoc positions:

- Fork this repository and make changes in `content/opportunities/_index.md` (there is a commented example showing where to add opportunities). Then push and create a PR/MR. Hamza or Pietro will merge it.

---

## Publications

If you want to add new papers:

- Simply update the `publications.bib` file located in the base directory. The GitHub workflow will take care of updating everything automatically.
